## 1. Introducción

El objetivo de esta práctica es realizar un análisis exploratorio y descriptivo de los datos de los pasajeros del Titanic,
aplicando operaciones de agregación, filtrado y visualización, siguiendo estrictamente las restricciones indicadas en clase.

En particular, el desarrollo se ha realizado **sin utilizar la librería pandas** y sin emplear notebooks interactivos,
implementando todo el flujo como un **programa Python reproducible**, apoyado en la librería **Polars** para el tratamiento
de datos.

## 2. Metodología y herramientas

- Lenguaje: Python
- Procesamiento de datos: Polars
- Visualización: matplotlib
- Organización: ejecución no interactiva mediante `python -m src.main`

Todas las tablas y figuras se generan automáticamente al ejecutar el programa, y se almacenan en las carpetas
`outputs/tables` y `outputs/figures` respectivamente.

## 5. Reproducibilidad

La práctica es completamente reproducible ejecutando el comando: python -m src.main

## 6. Uso de GitHub, GitHub Actions y GitHub Pages

Todo el desarrollo se ha almacenado en un repo GitHub (https://github.com/gtomaino58/titanic-polars-ci.git).
En ese repo se ha incluido un script (ci.yml) que cuando se realiza un push ejecuta una serie de instrucciones
que generan en GitHub el entorno necesario para ejecutar el código Python desarrollado, que genera las respuestas
a las preguntas planteadas por la practica (Ej 1 y Ej 2) (GitHub Actions) y finalmente estos resultados se ponen
a dispisicion del usuario en una landing para que los pueda consultar (GitHub Pages). 

## 7. Conclusiones

El análisis confirma la influencia de variables como la clase social, el sexo y la edad en la supervivencia
de los pasajeros del Titanic.

La práctica demuestra la viabilidad de realizar análisis de datos completos sin recurrir a pandas ni a entornos
interactivos, aplicando un enfoque reproducible y estructurado.

No usar pandas (y en consecuencia seaborn que esta basado en pandas) ha sido un requisito transmitido por el 
profesor en clase durante la presentación de la práctica.
