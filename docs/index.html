Automatización

El proyecto utiliza GitHub Actions para ejecutar el pipeline completo (Ejercicio 1 + Ejercicio 2) en un entorno Linux limpio, generando tablas, figuras e informes de forma reproducible.

Los resultados también se adjuntan como artifacts en cada ejecución del workflow.
